Tutorial

Creating a Webpage with SemanticUI

Now that we know the strengths and weaknesses of Semantic UI and Flask, let's create an effective webpage using them.

Start with Research

Before you get to coding, research is key. There are a lot of differnt tools unique to Semantic UI that are useful in creating a webpage. Specific names for classes in order to get a certain look are very common. Make sure you have a good understanding of your tools first.

The Head

Begin with the head tag of your page. This is where we do the ground work to allow both Semantic UI and Flask to be implemented within the page.


The 'stylesheet' link and script are what allow Semantic UI to be used. The code between the % symbols are generated through Flask.

UI Containers and Menus

Semantic UI has a container system that allows for easy division of content. One handy built in function is the 'ui top attached menu', which allows for the easy creation of a menu system to navigate your site.

Seperating Content

When writing up the content of your page, you will surely wish to organize your text in various more attractive ways. Semantic UI allows you to place different topics in different containers, allowing for different headers, bulleted lists, and other visually appealing options.


Semantic UI also has many options for images used in your webpage. Images can be centered or resized without the use of any CSS.

Final Touches

After you have everything you want on your page set up, finish it up with some nice design touches. Moving things around and adding divisions is easy with Semantic UI. Add a CSS stylesheet as well for some extra flair, and you're done!

Create a simple Flask App

Make a Flask app

Following the installation of Flask, create a file for running your website with a .py extension

Using your favourite editor, enter the following lines of code for your Flask app:

from flask import Flask, render_template
app = Flask (__name__)
@app.route('/')
def index():
     return render_template('index.html')
if __name__ == "__maine__":
     app.run(debug=True)

NOTE: index.html should instead be the name of your html page that you want Flask to run! (with the .html extension)

NOTE: inside 

app.run()
the parentheses can be empty as above, but we put: 
debug=True
for your convenience in developing - as you make changes and save them, you can simply refresh the page (localhost:5000) to view changes!

Run your Web App on Flask

Now, in your terminal, run the following command, where 

filename
is instead the chosen name of your flask app

python filename

On your terminal screen, you will be given a link to view your flask app. Copy paste it into your favourite browser, or simply put into your address bar: localhost:5000

This will run the flask app locally and you can see changes from your computer
