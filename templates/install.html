Installing Frameworks

Flask

You can refer to the documentation at the official site for Flask: http://flask.pocoo.org/docs/1.0/installation/

Install Python

First, you need to install Python from here (Latest version of Python 3 is recommended): https://www.python.org/downloads/

Install Flask

Once Python is installed, open your terminal.

Install Flask:

pip install flask

Install gunicorn:

pip install gunicorn

Congrats! You now have Flask installed. If you want to see how to make a simple Flask app, see the Tutorial page.

Semantic UI

You can refer to the documentation at the official site for Semantic UI: https://semantic-ui.com/

Note: Make sure all commands are run inside the project folder.

Install NodeJS

First, you need to install NodeJS from here: https://nodejs.org/en/

Install Gulp

Once NodeJS is installed, install gulp by running this command in the terminal:

npm install -g gulp

Install Semantic UI

Finally, install Semantic UI by running these commands in the terminal:

npm install semantic-ui --save
cd semantic/
gulp build

Heroku

Once you made your finished website, you can deploy it to Heroku

Make an account on Heroku

First, you need to make an account on Heroku: https://signup.heroku.com/

Create a list of your Python dependencies

You will need this for Heroku to know what packages your app has installed. In your terminal, after navigating to your Project's lcoation, run the command below

pip freeze > requirements.txt

Make a "Procfile"

This basically tells what processes to run on the application. In our case, we use gunicorn, so we do the following in the terminal...

vi Procfile

This will open vim, press i to begin typing the following into the file

web:gunicorn filename:app

NOTE: filename should instead be the name of your flask application (without the .py extension)

Now exit vim by hitting the escape key, and pressing WQ

Make an application on Heroku

Go to Heroku and create a new app

Give a unique app name and follow the remaining instructions

Install the Heroku CLI

This will be needed to continue in the terminal

Run Heroku create

log into heroku from your terminal like so:

heroku login

You will be prompted to log in from your default browser. Then run the following command:

heroku create

Setup Heroku

This is needed to be able to "push" your code on to heroku's servers

git init
heroku git:remote -a NAMEOFYOURHEROKUAPP

NOTE: replace the all caps part of the command with the name you gave your heroku app

Confirm that Heroku's servers are added with this command:

git remote -v

Final Step! Push your code onto Heroku

Run the following commands to push your code to heroku's servers and your website should now be online!

git add .
git commit -m "initial commit"

To confirm, see if heroku is available as an option with the command below:

git status
git push heroku master

CONGRATS! Your website should now be online with Heroku. After the results are done loading on your terminal screen, heroku should give you a link to view your website
